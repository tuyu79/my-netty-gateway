# 数据交互

## 和 base-service 的数据交互

### MQ

- 设备上行数据

| 名称      | 类型    | 长度 | 是否必须 | 描述         |
|:----------|:--------|:----:|:--------:|:-------------|
| channelId | Integer |  -   |    是    | 连接的通道id |
| msgType   | String  |  50  |    是    | 消息类型     |
| msgId     | Integer |  -   |    是    | 消息id       |
| mobile    | String  |  20  |    是    | 手机号       |
| msgBody   | String  |  -   |    是    | 消息体       |

- 平台下行数据

| 名称      | 类型    | 长度 | 是否必须 | 描述         |
|:----------|:--------|:----:|:--------:|:-------------|
| channelId | Integer |  -   |    是    | 连接的通道id |
| msgType   | String  |  50  |    是    | 消息类型     |
| mobile    | String  |  20  |    是    | 手机号       |
| msgBody   | String  |  -   |    是    | 消息体       |

- 连接断开上行数据

| 名称      | 类型    | 长度 | 是否必须 | 描述         |
|:----------|:--------|:----:|:--------:|:-------------|
| channelId | Integer |  -   |    是    | 连接的通道id |
